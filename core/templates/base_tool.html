Administrative Boundary Geocoder

A tool for geocoding text input to political administrative boundaries.

Built by the community and William & Mary geoLab, the Admin Geocoder is an online tool and database for geocoding textual name references to administrative boundaries (i.e., state, county) to their spatial representation in any country in the world. We currently search administrative unit names against a database of approximately 1 million boundaries within over 200 entities, including all 195 UN member states. All boundaries are available to view or download in common file formats, including shapefiles; the only requirement for use is acknowledgement.
